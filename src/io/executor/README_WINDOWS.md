# Исполнитель-X

## Запуск

Вы используете сборку Исполнителя под операционную систему Windows.
Для запуска нужно использовать скрипт executor.cmd.

Посмотрите подсказку, чтобы узнать о консольном интерфейсе Исполнителя-X:
```<путь>executor.cmd -h```


Из-за технических ограничений, используется кодировка Windows-1251 (cp1251) для отображения символов в консоли, изменять ее не рекомендуется.


Понадобится скрипт с кодом, который будет исполнен. Шаблонная реализация файла скрипта:

```
метод Скрипт(ПервыйАргумент: Строка, ЕщеОдинАргумент: Строка)
    // Ваш код здесь
; 
```
